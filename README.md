# dj-mooc
### Proof of Concept Django Restful API for E-learning (i.e. MOOC)

## RUN
First time
> pip install -r requirements.txt
> python manage.py makemigrations
> python manage.py migrate
When run server
> python manage.py runserver

## Test API Endpoint
http://localhost:8000/courses
